# RoadSafetyInsights MVP (`rsi_src`)

The core repository for the **RoadSafetyInsights** Minimum Viable Product. This platform transforms raw regional incident data into actionable driver awareness by aggregating local traffic reports, mapping high-risk road zones, and providing proactive safety alerts.

## Features

* **Data Aggregation Engine:** Automated Python scrapers that track, filter, and normalize historical accident/police data using localized keyword matching.
* **Geospatial Hazard Mapping:** Visualizes high-risk road segments and accident-prone clusters on an interactive web interface.
* **Proactive Awareness:** Built to supply drivers and fleets with localized, real-time safety insights to prevent future collisions.
